# sherpalhakpa.github.io
## Personal Portfolio Website Project

This is a small personal web design project I started to build my personal portfolio website.

1. Currently I am working on the very first version which will be a simple website with just HTML and CSS. Additional components will be added as I work to add more elements to the web page.

2. JQuery (https://github.com/jquery/jquery) and Bootstrap (https://github.com/twbs/bootstrap) was added for further responsiveness and cutting edge design

Following Plugins were used
----------------------------------------------------

3. Supersildes was used for the landing page. Downloaded from:
   https://github.com/nicinabox/superslides

4. Owl Carousel was used for responsive slider section for skills
   https://github.com/OwlCarousel2/OwlCarousel2

5. Easy Pie Chart used for skill percentage Chart
   https://github.com/rendro/easy-pie-chart

6. Typed.js used for landing page typing animation
   https://github.com/mattboldt/typed.js/
